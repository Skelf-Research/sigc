    Operators¶

    sigc includes 120+ built-in operators for signal construction.

    Operator Categories¶

    Category Count Description Link
    Arithmetic 15 Math operations abs, log, sqrt, pow
    Comparison 6 Relational operators >, <, >=, <=
    Logical 4 Boolean logic and, or, not, where
    Data Handling 8 Missing data, cumulative fill_nan, cumsum
    Time-Series 20+ Per-asset over time ret, lag, rolling_*
    Cross-Sectional 12 Across assets zscore, rank, winsor
    Technical 4 Technical indicators rsi, macd, atr
    Portfolio 3 Weight construction long_short, neutralize

    Quick Reference¶

    Most Common Operators¶

    Text Only
    // Returns
ret(prices, 20)           // 20-day return

// Time-series
lag(x, 5)                 // Lag by 5 periods
rolling_mean(x, 20)       // 20-day moving average
rolling_std(x, 20)        // 20-day rolling std

// Cross-sectional
zscore(x)                 // Standardize
rank(x)                   // Rank 0-1
winsor(x, p=0.01)         // Clip outliers

// Portfolio
rank(signal).long_short(top=0.2, bottom=0.2)

    Operator Signatures¶

    Each operator has a signature:

    Text Only
    operator(input1: Type, input2: Type, ...) -> OutputType

    Example:

    Text Only
    ret(x: Panel, n: Scalar) -> Panel
zscore(x: Panel) -> Panel
rolling_mean(x: Panel, window: Scalar) -> Panel

    Using Operators¶

    Basic Usage¶

    Text Only
    signal example:
  returns = ret(prices, 20)
  normalized = zscore(returns)
  emit normalized

    Chaining¶

    Text Only
    signal chained:
  step1 = ret(prices, 20)
  step2 = zscore(step1)
  step3 = winsor(step2, p=0.01)
  emit step3

    Composition¶

    Text Only
    signal composed:
  emit winsor(zscore(ret(prices, 20)), p=0.01)
